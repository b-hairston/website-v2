## Overview

This project is designed to analyze and visualize Samsung Health data using tools like Python, Prefect, Postgres, and Metabase. 


## Key Features

- **ETL Pipelines**: Built with Prefect for automated data extraction, transformation, and loading.
- **Health Data Analysis**: Supports metrics like step count, exercise, heart rate, and sleep.
- **Metabase Integration**: For visualization and business intelligence.

## Technologies Used

- **Python**: Primary programming language for data processing.
- **Prefect**: For managing ETL pipelines.
- **Postgres**: For data storage.
- **Metabase**: For business intelligence and data visualization.

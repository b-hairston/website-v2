## Overview

BeanBot is a custom Discord bot designed to add interactive and fun features to your Discord server. It provides both music playback and meme functionalities, as well as several custom commands for unique server interactions.

## Overview

BeanBot is a Discord bot that allows for user interaction, helping automate and enhance the experience within a server. It is designed to handle commands related to memes, fun messages, and administrative tasks, all packaged in an easy-to-use bot format.

## Key Features

- **Custom Commands**: BeanBot allows users to create specific commands, enhancing the server experience.
- **Command Customization**: The bot is flexible and allows for the creation of custom behavior, such as sending memes or interacting with users in unique ways.
- **Fun & Interactive**: Includes commands related to memes and fun responses for a more interactive user experience.
- **Server-Specific Use**: Built initially for a personal use case, BeanBot supports interactions that can be tailored to specific Discord servers.

## Technologies Used

- **Python**: BeanBot is written in Python, a popular language for creating Discord bots.
- **Discord.py**: The bot utilizes the `discord.py` library for interacting with the Discord API, enabling smooth communication with users.


## Challenges and Learnings

- **Customizability**: One of the challenges was making the bot flexible enough for personal use while allowing others to use and modify it for their needs.
- **Interaction**: Ensuring that commands were user-friendly while allowing for creative uses in different servers was a key learning point.

## Outcome

BeanBot serves as a personalized, customizable tool that integrates well with Discord servers, offering both entertainment and utility features for server members.The bot is officially retired and not currently in use

## Overview

Tumblr Blocker is a utility that automates the transfer of blocked Tumblr blogs from one account to another. This tool is designed for users who manage multiple Tumblr accounts and need an efficient way to maintain their block lists across these accounts.
Key Features

- Automated Block List Transfer: Allows users to seamlessly transfer blocked blogs between different Tumblr accounts without having to manually re-block each account.
- Ease of Use: Provides a simple interface to manage and sync block lists, saving time for users with multiple profiles.
- Efficiency: The tool automates the process, reducing the need for repetitive tasks and ensuring blocked accounts are consistently handled across accounts.

## Technologies Used

- TypeScript: The tool is built with TypeScript for strong typing and maintainability.
- TailwindCSS: Styling and theme
- Vercel: Hosting & CI/CD

## Challenges and Learnings

One of the challenges faced in developing the Tumblr Blocker was ensuring that blocked accounts were accurately transferred between accounts, especially when handling large numbers of blocked profiles. This involved setting up a robust backend to handle the synchronization between accounts efficiently.
Outcome

The Tumblr Blocker has made it easier for users to manage their Tumblr block lists across multiple accounts, improving efficiency and reducing the manual effort required to maintain content control on the platform.
